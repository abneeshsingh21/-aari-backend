## ğŸ” Troubleshooting

### "Wake word not detected"
```
1. Check Background Mode is enabled
2. Disable battery optimization
3. Restart the app
4. Speak clearly and loudly
```

### "Can't connect to backend"
```
1. Check Heroku app is running:
   heroku logs --tail --app your-app-name
   
2. Verify Heroku URL is correct in ApiClient.java

3. Test manually:
   https://your-app-name.herokuapp.com/api/health
   (Should return: {"status":"healthy"})
```

### "App crashes on startup"
```
1. Clear cache: Settings â†’ Apps â†’ AARI â†’ Storage â†’ Clear Cache
2. Reinstall the app
3. Ensure phone has 100MB+ free storage
```

### "No audio output"
```
1. Check volume is not muted
2. Test speakers with other app
3. Restart phone
```

### "Works on WiFi but not 4G"
```
1. Check Heroku app is public (not private)
2. Verify URL is correct
3. Some carriers block certain ports - use Heroku (port 443 is standard)
```
