## ğŸ“‹ Quick Steps (10 minutes)

### Step 1: Create GitHub Account (Free)
1. Go to: https://github.com/signup
2. Sign up with email
3. Create account

### Step 2: Create GitHub Repository
1. Go to: https://github.com/new
2. Name: `aari-backend`
3. Description: `AARI Voice Assistant Backend`
4. Select: **Public** (important!)
5. Click: **Create repository**

### Step 3: Push AARI Code to GitHub
```powershell
cd "c:\Users\lenovo\Desktop\aari\VoiceAssistant\backend"

# Configure Git
git init
git config user.email "your-email@gmail.com"
git config user.name "Your Name"

# Add and commit
git add .
git commit -m "AARI Backend"

# Add remote (using your GitHub username: abneeshsingh21)
git remote add origin https://github.com/abneeshsingh21/aari-backend.git
git branch -M main
git push -u origin main

# It will ask for your GitHub username and password (or use personal token)
```

### Step 4: Create Render Account
1. Go to: https://render.com
2. Click: **Sign up**
3. Select: **Sign up with GitHub**
4. Authorize Render to access your GitHub

### Step 5: Deploy to Render
1. Go to: https://dashboard.render.com
2. Click: **New** (+ button, top right)
3. Select: **Web Service**
4. Choose: `aari-backend` repository
5. Fill in:
   - **Name:** `aari-backend`
   - **Runtime:** `Python`
   - **Region:** (any)
   - **Build Command:** 
     ```
     pip install -r requirements.txt && python -m spacy download en_core_web_sm
     ```
   - **Start Command:** 
     ```
     gunicorn app:app
     ```

### Step 6: Add Environment Variables
1. Scroll to **Environment** section
2. Add these variables (from your `.env` file):
   - `GOOGLE_API_KEY` = (your key)
   - `SENDER_EMAIL` = (your email)
   - `SENDER_PASSWORD` = (your app password)

3. Click: **Create Web Service**
4. Wait 2-3 minutes for deployment

### Step 7: Get Your URL
1. On Render dashboard, click your service
2. Copy the **Service URL** (looks like: `https://aari-backend.onrender.com`)

### Step 8: Update Android App
Edit: `android/src/main/java/com/voiceassistant/app/ApiClient.java`

Find:
```java
private static final String CLOUD_BACKEND = "https://aari-backend.onrender.com/api";
```

Replace `aari-backend` with your actual service name from Step 7.

