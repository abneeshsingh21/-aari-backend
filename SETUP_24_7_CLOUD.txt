### Step 3: Ensure API Keys are Set
Edit: `backend/.env`
```
GOOGLE_API_KEY=your_key_here
SENDER_EMAIL=your_gmail@gmail.com
```

### Step 4: Deploy to Heroku
```powershell
cd "c:\Users\lenovo\Desktop\aari\VoiceAssistant"
.\DEPLOY_TO_HEROKU.bat
```

Follow the prompts. It will:
1. Log you into Heroku
2. Create your app
3. Deploy code
4. Give you a URL like: `https://aari-voice-assistant.herokuapp.com`

### Step 5: Update Android App
**File:** `android/src/main/java/com/voiceassistant/app/ApiClient.java`

Change:
```java
private static final String BASE_URL = "http://192.168.1.100:5000";
```

To:
```java
private static final String BASE_URL = "https://aari-voice-assistant.herokuapp.com";
```

Replace `aari-voice-assistant` with your actual app name from Step 4.
